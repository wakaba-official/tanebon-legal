# tanebon-legal

iOS アプリ **「たねぼん」**(コードネーム: tanebon)の公開ポリシー類を配信する
GitHub Pages リポジトリです。

## 公開 URL

- プライバシーポリシー: <https://wakaba-official.github.io/tanebon-legal/privacy/>
- サポート: <https://wakaba-official.github.io/tanebon-legal/support/>

## 構成

- `privacy.md` — プライバシーポリシー本文
- `index.md` — トップページ(各ポリシーへのリンク)
- `_config.yml` — Jekyll 設定(GitHub Pages のテーマ等)
- `support.md` — サポートページ本文

## 更新

本リポジトリの内容は GitHub Pages により自動でビルド・配信されます。
編集は `privacy.md` を直接更新してコミットしてください。
